# 성경·찬송 (BibleApp)

성경을 읽고, 찬송가·복음성가 가사를 직접 등록해 보관할 수 있는 안드로이드 앱입니다.

## 기능

- **성경 읽기** — KJV(King James Version) 66권 전체, 1,189장 수록
  - 구약/신약 책 목록, 책 이름 검색
  - 장 단위 가로 스크롤 이동, 이전/다음 장 버튼
  - 글자 크기 6단계 조절
  - 구절 탭 → 클립보드 복사
  - 구절 길게 누르기 → 북마크 토글
- **찬송가/복음성가** — 사용자가 제목과 가사를 직접 입력해 등록
  - 목록 검색, 편집, 삭제
- **북마크** — 저장한 구절 모아 보기, 탭하면 해당 위치로 이동

- **주보 요약 전용 AI 지정**: 일반 구절 AI와 별도로, 주보 요약(이미지 인식)에 쓸 제공자/모델을 따로 선택. Gemini 비전 기본, Groq 비전 모델(llama-4-scout 등)도 선택 가능.

## v2.5.0 — AI 연동 (Gemini)

- **설정 > AI 설정**: 제공자/모델 선택(현재 Gemini, 추후 Groq 대비), **API 키 입력 → 테스트 & 저장**(앞 4글자만 남기고 *로 마스킹), **구절 프롬프트**·**주보 요약 프롬프트** 입력.
- **읽기 화면 구절 메뉴 > ✨ AI에 보내기**: 단일 또는 다중복사 범위의 구절을 사용자 프롬프트와 함께 Gemini에 전송, 결과 표시. (복사 가능)
- **AI 메모 저장**: 결과가 좋으면 구절에 AI 메모로 저장. 기존 AI 메모가 있으면 **덮어쓰기 전에 '기존 내용 확인'** 후 결정. **AI 메모 보기/수정**으로 내용 보완 가능.
- **표시**: 사용자 메모(주황) 점 아래 **AI 메모는 연두색 점**. 여러 구절 AI 메모는 **첫 구절**에 표시.
- **내보내기(.json)**: 파일명 `영한Bible_연월일_시분초.json`. 프롬프트·AI 메모는 항상 포함, **API 키는 내보낼 때 포함 여부를 물어봄**. 가져오기 시 함께 복원.
- TTS 아이콘을 토글(B·⇄)과 통일된 **원형 'TTS' 레터마크**로 정리.
- (예정) 주보 자동 인식 → 주보 요약 프롬프트로 구절 정리/요약.

- **단락 제목 직접 입력(✎ 제목)**: 각 단락에 원하는 제목을 붙이거나 지울 수 있습니다(큐레이션 요약은 보존).
- 사용자 지정 단락은 기기에 저장되며 우선순위는 사용자 지정 > 번들 자료 > 장 전체.

## v2.3.1 — QT 모드 개선 (단락별 묵상·진입점·단락 자료)

- **진입점 이동**: QT 모드 토글을 상단바("성경" 제목 오른쪽)로 옮겨 잘림 없이 눈에 띄게 표시(켜짐 시 금색 `QT ●`).
- **단락별 묵상·기도**: 한 장을 단락으로 묶고, **단락마다** 묵상·기도 입력과 사진(첨부·회전·삭제)을 따로 기록.
- **단락 자료 확장**: 마가복음 1~6장을 표준 단락 구분(예: 씨 뿌리는 자의 비유, 오천 명을 먹이심 등)으로 그룹화하고 자체 요약을 제공. 단락 자료가 없는 장은 의미 없는 제목을 붙이지 않고 절 범위만 표시.
- **중복 표기 제거**: 폴백 단락의 "1–10절 · 1–10절" 중복을 제거.
- **두란노 링크 제거**.
- 백업/CSV는 단락 단위로 반영(qt.csv = book,chapter,range,meditation,prayer,qtDate). 사진 경로는 가져오기 시 자동 보정.



- **QT 모드** — 성경 화면 상단 `QT` 버튼으로 켜고 끔. 켜면 장 선택 시 묵상 화면이 열리고, 끄면 일반 성경 보기로 즉시 복원(QT 기록은 보존).
  - 장을 단락으로 묶어 **단락 제목 + 간단한 요약**(자체 작성, 일부 장 선제공 / 나머지는 절 단위 자동 구분)과 함께 표시.
  - **묵상**·**기도** 체크 시 각각 입력칸이 열리고, 저장 시 그 시점이 **QT 날짜**로 기록.
  - 장 칩에 **✓** 표시로 QT 완료 여부를 한눈에, 상단에 "최근 QT 날짜" 표기.
  - 사진 첨부(갤러리·카메라)·회전·삭제, 기록 수정·삭제 지원.
  - "오늘의 말씀 (두란노 QT)" 링크 — 본문은 담지 않고 해당 날짜 페이지를 브라우저로 엽니다(저작권 보호).
- **데이터 백업/가져오기** (설정)
  - **백업 내보내기(.zip)**: 설정·메모·QT·사진 전체 → 재설치 후 복원용.
  - **CSV로 내보내기**: 메모·QT 텍스트(엑셀용, BOM 포함, 사진 제외).
  - **가져오기**: 백업 zip 선택 → 메모·QT·사진·설정 복원(사진 경로 자동 보정).
- 안정성: 모든 저장/파일/압축 작업 예외 처리, 내보내기·가져오기는 백그라운드 처리. 새 의존성 없음.

> 참고: 생명의삶/두란노 등 외부 QT의 단락 제목·요약·본문은 저작권이 있어 포함하지 않으며, 단락 요약은 자체 작성한 원문입니다. `assets/qt_outline.json`에 장을 추가해 확장할 수 있습니다.



- **구절 누름 동작 사용자화** — 설정 ▸ "구절 누름 동작"에서 한 번 / 두 번 / 세 번
  연속 누름 / 꾹 누름(롱프레스) 각각에 동작을 지정합니다.
  - 동작: 없음 · 팝업 메뉴 · 다중 복사 · 북마크 · 한 구절 복사
  - 기본값: 한 번=없음(의도치 않은 팝업 방지), 두 번=팝업 메뉴, 세 번=다중 복사, 꾹=북마크
  - 북마크 토스트의 별(★) 기호 제거
- **다중(연속) 구절 복사** — 시작 구절에서 동작 실행 시 시작점 기억(토스트),
  같은 장의 끝 구절에서 다시 실행하면 그 구간을 한 번에 복사(토스트). 100구절 초과 시 100구절까지.
  - 팝업 메뉴 제목 우측의 **다중복사** 버튼으로도 시작/완료 가능(활성 시 "여기까지 복사"로 표시)
  - 시작 구절은 옅은 색으로 하이라이트
- **메모 장표 글자 선택 복사** — 공유 미리보기에서 구절·메모 텍스트를 선택해 복사 가능
- **패드 본문 폭 확대** — 넓은 화면에서 좌우 여백을 최소화해 본문이 화면 폭을 충분히 활용
- **공유 서명 문구** — 설정에서 장표에 붙일 문구(예: 이름)를 지정. 비우면 기존대로 "한영Bible"
- **카메라 직접 촬영 첨부** — 메모에 갤러리 외 카메라 촬영 사진을 바로 첨부(EXIF 회전 자동 보정)
- **라이트(일반) 모드** — 설정 ▸ 화면 테마에서 시스템 / 라이트 / 다크 선택(기본 다크)
- 팝업 메뉴 "인터넷 검색" 글자색을 다른 항목과 동일하게 통일



폰과 패드를 **하나의 APK**로 동시에 지원합니다. 기기의 `smallestScreenWidthDp`
(600dp 이상)를 런타임에 감지해 자동으로 레이아웃을 전환하므로, 폰에서는 기존
동작이 그대로 유지되고 패드에서는 넓은 화면에 맞춰 다음이 적용됩니다.

- **성경 본문** — 패드에서는 한 줄 길이를 읽기 좋은 폭(약 720dp)으로 제한하고
  가운데 정렬해, 가로로 늘어져 읽기 힘든 문제를 없앴습니다.
- **책 목록** — 패드에서 2열(와이드 화면은 3열) 그리드로 표시해 66권을 한눈에.
- **메모 목록** — 패드에서 2~3열 카드 그리드로 표시.

### 메모 장표 — 디자인 템플릿 6종 + 폰/패드 공유

메모 뷰어를 한 장표(이미지 카드)로 보여주는 기능을 다양화했습니다.

- 상단 칩에서 장표 디자인을 즉시 전환: **클래식 / 종이 / 미니멀 / 나이트 / 하늘 / 구절강조**
  (마지막 선택은 자동 저장됩니다)
- **공유 시 형식 선택** — "공유"를 누르면 **폰용(세로 1080px)** / **패드용(가로 1600px)**
  을 고를 수 있고, 선택한 폭으로 고정 렌더링하여 어느 기기에서도 동일한 결과물이
  만들어집니다. 패드용은 사진을 2열 그리드로 배치합니다.
- 공유 이미지는 기기 화면을 그대로 캡처하지 않고 화면에 붙지 않은 별도 카드를
  목표 픽셀 폭으로 그려 PNG로 내보내므로 글자·여백 비율이 일정합니다.


## 빌드 방법

1. Android Studio (Hedgehog 이상 권장) 설치
2. 이 폴더를 Android Studio에서 `Open` 으로 엽니다
3. Gradle 동기화가 끝나면 상단 ▶(Run) 버튼으로 실행하거나,
   터미널에서 다음 명령으로 APK를 생성합니다:

   ```
   ./gradlew assembleDebug      # macOS / Linux
   gradlew.bat assembleDebug    # Windows
   ```

4. 생성된 APK 위치:
   `app/build/outputs/apk/debug/app-debug.apk`

   기기에 설치하려면:
   ```
   ./gradlew installDebug
   ```
   또는 `app-debug.apk` 파일을 안드로이드 기기로 옮겨 직접 설치합니다.
   (기기 설정에서 "출처를 알 수 없는 앱 설치" 허용 필요)

- **성경 본문**: King James Version. 퍼블릭 도메인이라 자유롭게 배포·사용 가능합니다.
- **한국어 성경 번역본**(개역개정·새번역 등)은 대한성서공회 등이 저작권을
  보유하고 있어 이 앱에 포함하지 않았습니다.
- **찬송가/복음성가 가사**는 앱에 기본 수록하지 않습니다. 사용자가 직접
  입력하며, 입력하는 가사에 대한 저작권 책임은 사용자에게 있습니다.
  21세기 찬송가 등 저작권이 있는 가사를 무단으로 배포하지 마세요.

이 앱을 마켓에 배포하려면 `applicationId`, 앱 이름, 아이콘을
본인 것으로 바꾸고, 서명 키로 release 빌드를 하면 됩니다.
